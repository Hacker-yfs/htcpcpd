HTCPCPD

Une implémentation du « Hyper-Text Coffee Pot Control Protocol », faits par des étudiants du Cégep de Sainte-Foy

Bienvenue sur le site du projet HTCPCPD ! HTCPCPD est une implémentation prototype du protocole HTCPCP. Ce site vous expliquera le but du projet HTCPCPD et la démarche que nous avons suivie pour arriver à notre prototype.

Qu'est-ce que HTCPCP ?

Toutes les années, l'IETF publie une RFC humoristique pour le poisson d'avril. En 1999, la blague de l'année fut le protocole HTCPCP (HyperText Coffee Pot Control Protocol), un protocole permettant de contrôler une cafetière à travers internet.

Pourquoi ?

Pourquoi ? Eh bien parce que nous sommes des geeks ! Avez-vous besoin d'une raison de plus ? ;) Dans le cadre de notre cours « Nouvelles technologies », nous devions choisir un sujet qui nous permettrait d'explorer le fonctionnement d'une technologie qui nous était inconnue. Prérequis et Installation

Pour utiliser HTCPCPD vous-même, veuillez vous référer au fichier README disponible avec le code source du projet

Utilisation

Serveur

  • Branchez la carte Arduino de relié à la cafetière sur un port USB du serveur
  • Modifiez la configuration du fichier htcpcpd.ini selon vos besoins
  • Partez le serveur avec la commande suivante:
{% highlight bash %} htcpcpd -c htcpcpd.ini {% endhighlight %}

Client

Étant donné que les navigateurs web modernes ne supportent pas encore le protocole HTCPCP, nous avons implémenté un client en ligne de console que vous pouvez partir avec la commande suivante :

{% highlight bash %} htcpcpd-client localhost 8000 {% endhighlight %}

Changez la valeur de localhost et 8000 par l'adresse et le port approprié si vous avez modifié la configuration par défaut du serveur.

Une fois dans le client console, vous pouvez taper la commande 'help' pour avoir une liste des commandes disponibles

CoffeePot> help

Documented commands (type help ):
========================================
EOF  brew  exit  pot  quit  resume  status  stop  water

Pour avoir de l'information sur une commande, tapez 'help' suivi du nom de la commande. 

CoffeePot> help pot
 
Show if the pot is in the machine       or not.
